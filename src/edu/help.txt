Как создать новый курс:
  - добавить класс в src/edu/course.js. в параметрах прототипа указать id и название
  - перевести название в модулях src/lang/ru.js и src/lang/en.js
  - добавить класс в return модуля src/edu/course.js
  - добавить создание нового класса в src/edu/courselist.js
  - создать в src/edu папку с названием курса для переводов упражнений

Как создать новое упражнение:
  - в нужный класс в src/edu/course.js добавить создание нового упражнения.
    параметрами конструктора указать название упражнения и id картый в папке
    src/battle-city/maps/createbot
  - добавить в src/lang/ru.js и src/lang/en.js по два ключа: <name> и <name>-desc.
