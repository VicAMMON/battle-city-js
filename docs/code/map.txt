==== Структура для размещения объекта? ====
Как лучше хранить объекты располагающиеся на карте? Сейчас объекты храняться в
виде списка. Минус такого подхода в том, что для поиска пересечени приходится
перебирать все объекты. Хранить объекты можно в виде дерева, либо просто в виде
трехмерного массива. Но вариант с массивом не выглядит привлекательно, даже
учитывая что js не выделяет память под пустые элементы массива, потому что танк
размером 32*32 будет занимать слишном много места, и операции по перемещению
объектов будут очень громоздкими.
