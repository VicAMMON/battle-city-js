==== Синхронизация данных с клиентами ====

Синхронизация полагается на несколько подсистем:
 - база данных для объектов (создание, получение по id и освобождение объектов),
 - клиентская база данных проксирующая запросы на сервер,
 - коллекции объектов и пакетная рассылка изменений на клиенты для экономии кол-ва запросов

Сериализация композитных объектов предполагает замену подобъектов на id на
    сервере, и получение подобъектов на клиенте через клиентскую БД (в БД или
    уже будут данные или она сделает запрос на сервер).
